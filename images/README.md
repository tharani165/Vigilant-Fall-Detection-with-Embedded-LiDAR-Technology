# Images

Visual assets for the Vigilant fall detection project, organized by category.

## `diagrams/`

- `block_diagram.png` — system block diagram (sensing → ML processing → fall decision → camera confirmation → notification)
- `workflow.png` — end-to-end process flow chart

## `app/`

Screens from the **PatientConnect** companion app prototype (caretaker/doctor/patient management):

- `patient_connect_overview.png` — login, patient list, and patient detail view
- `patient_connect_users.png` — caretaker/user list
- `patient_connect_doctors.png` — doctor directory
- `patient_connect_edit_patient.png` — patient record edit form
- `login_screen_prototype.png` — an earlier login screen prototype

## `results/`

Demonstration captures from prototype testing:

- `test_scenario_layout.png` — room layout used for simulated fall test scenarios
- `notification_banner.png` — fall-alert push notification on a phone lock screen
- `alert_message.png` — SMS alert message thread
- `fall_alert_with_capture.png` — SMS alert including the captured confirmation image of a detected fall

These are referenced directly from the main [README.md](../README.md).
